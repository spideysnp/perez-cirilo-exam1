# CIS 2336 - Exam 1 Starter Files

This repository contains the starter files for Exam 1.

## Files Included

```
exam1_FirstName_LastName.html
styles.css
script.js
```

## Instructions

1. Clone this repository to your computer.

2. Create your own GitHub repository named:

```
lastname-firstname-exam1
```

Example

```
aly-heba-exam1
```

3. Copy these starter files into your own repository.

4. Complete the exam.

5. Commit your changes regularly.

6. Push your final work to GitHub.

7. Submit **only your own GitHub repository link** on Canvas.

Good luck!
